# Climalert

## Tecnologías

- Java 21
- Spring Boot
- Spring Data JPA
- H2 Database
- JavaMailSender
- WeatherAPI

## Variables de entorno

Para ejecutar el proyecto configurar:

- WEATHER_API_KEY
- MAIL_USERNAME
- MAIL_PASSWORD

## Base de datos

Se utiliza H2.

La consola se encuentra en:

http://localhost:8080/h2-console
